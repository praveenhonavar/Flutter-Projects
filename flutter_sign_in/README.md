# flutter_sign_in

A new Flutter project.

## Dependencies

Add this in android **dependicies**
```
        classpath 'com.google.gms:google-services:4.3.3'
```


Add this in android **app/src**
```
apply plugin: 'com.google.gms.google-services'

```



Add this in **pubspec.yaml**

```
firebase_auth:
google_sign_in:

```
